# Pneumonia Detection using CNN

This project detects pneumonia from chest X-ray images using a Convolutional Neural Network (CNN).

## Dataset
The Chest X-Ray Images (Pneumonia) dataset is taken from Kaggle.

ðŸ”— https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia

## Model
- Input size: 224x224 RGB images
- CNN with 3 convolutional blocks
- Binary classification (NORMAL / PNEUMONIA)
